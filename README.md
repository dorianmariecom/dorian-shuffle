# `dorian-shuffle`

Shuffles the lines of the input

e.g. `ls | shuffle`

### Install

```bash
gem install dorian-shuffle
```

Or as part of my other gems:

```bash
gem install dorian
```

### Usage

From my history:

```bash
cat ~/ping-pong-members.txt | shuffle
printf 'Dorian\nMadiha\nRémi\nRayane\nMélanie\nJade' | shuffle
```
